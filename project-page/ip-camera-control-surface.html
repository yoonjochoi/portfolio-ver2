IP Camera Control Surface

2012

network

Group Project: Bruna Calheiros, Mark Breneman, Yoonjo Choi

IP Camera Control Surface comes from a concept of driving IP cameras that are installed throughout the ITP floor. Using the driving stick you can select and change the camera you wish to view. Turning the wheel will interact with the panning of the camera, while the window wiper controls the tilting. On the end of the window wiper is another controller which focuses the camera. There are also two pedals, each controlling the zooming (right pedal) and the iris (left pedal).

Software: Arduino, Node JS, Javascript
